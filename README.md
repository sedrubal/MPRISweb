DBusMediaWeb
============

Control your musicplayer through the web.

Usage
-----

 - Install: `./setup.py install`
 - Run the app: `./main.py`
 - Go to: [http://localhost:8888/](http://localhost:8888/)

Features
--------

 - dbus
 - websockets
 - python3
 - tornado
 - Bootstrap
 - Javascript
